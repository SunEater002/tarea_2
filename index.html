    Presentacion

    El **desarrollo web** es la creación y mantenimiento de sitios web, abarcando dos áreas principales: 1. **Front-end**: La parte visual con la que interactúan los usuarios 2. **Back-end**: La parte que gestiona el servidor, bases de datos y lógica, utilizando lenguajes como **Python** o **PHP**.

    Font-end

    El front-end es la parte del desarrollo web que se encarga de la interfaz y experiencia del usuario. Involucra todo lo que el usuario ve y con lo que interactúa en un sitio web o aplicación, como el diseño, la estructura, los botones, menús, imágenes y textos.

    Back-end

    El back-end es la parte del desarrollo web que gestiona la lógica del servidor, bases de datos y procesamiento detrás de escena. Es responsable de que las funciones de un sitio web o aplicación funcionen correctamente, como el manejo de usuarios, procesamiento de datos, y autenticación.
